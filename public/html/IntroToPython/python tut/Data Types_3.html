Welcome to lesson number four, data types! Today, we are going to learn about three data types: Strings, Ints and Floats. Let's get started 😃.

Strings:

You might not know it, but we've actually already seen an example of a string 😁, "hello world". You might feel a little confused right now 😵‍💫, or maybe you understood everything just fine 😄, but don't worry I'll explain ether way 😉. A string is just a pice of text, like hello world, and for Python to be able to distinguish it from code you need to put double "" or single '' quotes around it, like so: "hello world"

Ints:

Int is short for integer, and integer just means a whole number. 1 is an integer. You can simply type a whole number and Python will consider it an integer, you don't need any quotes. Please note that if you put a number inside quotes, it will not be read as an integer but as a string, so be careful 😉.

Floats:

A float is a floating point number, an example of a float is 3.14 because it's a number with one or more decimal places. Floats work the same way as ints, they don't need to be put in quotes and if you do they become strings.

Thats it for today's lesson! You now know three very important important data types of Python 🎊, see you soon.
