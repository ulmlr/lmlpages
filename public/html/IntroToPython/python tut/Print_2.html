To better explain the basics of Python, in this lesson we are going to be using one of Python's built in functions, print(). What is a function though 🤔? A function is a block of code that has already been written for you and that can be executed with a single line of code.

To execute a function you need three things:

  1. The Name of the function
  2. Brackets () - you need the brackets to call the function, without them your only referencing it
  3. And Arguments - any extra information the function needs

They are used in this order/way 👇:

NameOfFunction(Arguments)

copy

Lets try this with print(), the name of the function is print and it takes one argument, the print function will then take the argument and print it in the console.
